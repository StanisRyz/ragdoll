# Adventurers technical report

Curated GLB characters: Barbarian, Knight, Mage, Ranger, Rogue, and Rogue Hooded.

Their source is `KayKit_Adventurers_2.0_FREE/Characters/gltf/`; runtime copies are in this directory. The available animation libraries are `Rig_Medium_General.glb` and `Rig_Medium_MovementBasic.glb` in `../animations/`. They are expected to target the package's Rig_Medium skeleton. No character scene, controller, AnimationTree, gameplay rig, or ragdoll is created in this stage; scale, skeleton compatibility, and material assignment must be confirmed in the forthcoming Character Showcase.
